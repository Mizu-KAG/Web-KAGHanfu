Thư mục images dùng để chứa ảnh website.

Bạn có thể thay các file placeholder SVG bằng ảnh thật của bạn.
Gợi ý đặt tên:
- logo.png
- product-1.jpg
- product-2.jpg
- product-3.jpg
- product-4.jpg
- product-5.jpg

Nếu đổi đuôi ảnh từ .svg sang .jpg/.png thì nhớ sửa lại đường dẫn trong index.html.
Ví dụ:
<img src="images/product-1.jpg" alt="Tên sản phẩm">
